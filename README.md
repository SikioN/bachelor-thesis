# Выпускная квалификационная работа (бакалавриат)

Этот репозиторий содержит материалы бакалаврской работы.

## Информация о работе
* **Автор:** Муравья Никита
* **Руководитель:** Бойцев Антон Александрович

## Структура проекта
* `main.tex` — основной файл LaTeX.
* `chapters/` — директория с главами работы.
* `references.bib` — список литературы.
* `vkr.sty` — стилевой файл оформления.

## Сборка
Для сборки работы в PDF-формат используйте `latexmk` или любую другую среду подготовки документов LaTeX.
